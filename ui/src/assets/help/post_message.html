You can get to the post message page by clicking on the "New Post" button from the home page.

First select the course you would like to post the thread to. Next type a title for the thread.

This page allows you to select a message type. Question threads ask a question, Note threads send a message, and poll threads allow users to vote on topics.

Once you have selected the type of post you want, you just type it into the body of the page just below all of the formating icons. You can use these icons to change the appearance of the message. These formatting icons work the same as they do in most word processing programs.
